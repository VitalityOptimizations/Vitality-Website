What is IMOD and should I run it every session?

+

IMOD (Interrupt Moderation) detects all USB host controllers on your system and disables interrupt moderation and coalescing on each one. This removes the artificial batching delay that USB drivers introduce, giving your mouse, keyboard, and controller the most direct signal path possible to Windows. Yes — run IMOD once every time you start a gaming session for lowest USB latency. It resets after a reboot.

What Quick Fixes does Vitality include?

+

Quick Fixes are one-click system repair tools in the Miscellaneous tab: Reset Network resets Winsock, flushes DNS, and resets the IP stack. Reset Windows Update repairs broken or stuck Windows Update. System Corruption Scan runs DISM and SFC to detect and fix corrupted Windows system files.
